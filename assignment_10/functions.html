LOGICAL OPERATIONS IN SQL

SQL Logical Operations

Logical operators are used to combine multiple conditions in a SQL query. Here are some common SQL logical operators:

AND

The AND operator is used to combine multiple conditions, and all conditions must be true for a record to be included in the result set.

SELECT * FROM EMPLOYEE
WHERE SALARY < 10000 AND AGE > 25;

OR

The OR operator is used to combine multiple conditions, and at least one condition must be true for a record to be included in the result set.

SELECT * FROM STUDENTS
WHERE RANK > 20 OR MARKS > 600;

NOT

The NOT operator is used to negate a condition. Records will be included in the result set if the condition is not true.

SELECT * FROM EMP
WHERE NOT SALARY = 10000;

Ordering Operation

Ordering operations sort the results of a query based on specified columns:

ASC

Sorts the results in ascending order (default).

SELECT * FROM EMP
ORDER BY ID ASC;

DESC

Sorts the results in descending order.

SELECT * FROM EMP
ORDER BY ID DESC;

Grouping Operation

Grouping operations are used to group rows that have the same values in specified columns into summary rows:

SELECT FNAME, LNAME FROM EMP
GROUP BY Name;

Aggregate Functions

Aggregate functions perform a calculation on a set of values and return a single value. Here are some common aggregate functions:

AVERAGE

SELECT AVG(salary) FROM EMP;

MINIMUM

SELECT MIN(salary) FROM EMP;

MAXIMUM

SELECT MAX(salary) FROM EMP;

SUM

SELECT SUM(salary) FROM EMP;

COUNT

SELECT COUNT(id) FROM EMP;

DISTINCT

SELECT DISTINCT(name) FROM EMP;

ROUND

SELECT ROUND(MARKS, 2) FROM STUDENTS;

String Functions

String functions are used to manipulate and retrieve information from string data types:

LENGTH

SELECT LENGTH('APPLE');

REVERSE

SELECT REVERSE('APPLE');

REPLICATE

SELECT REPLICATE('APPLE', 3);

UPPERCASE

SELECT UPPER('apple');

LOWERCASE

SELECT LOWER('APPLE');

CONCATENATION

SELECT CONCAT('NE', 'PA', 'L');

LEFT MOST CHARACTER

SELECT LEFT('APPLE', 3);

RIGHT MOST CHARACTER

SELECT RIGHT('APPLE', 3);

Others

Additional SQL functions and operators:

IN

SELECT ADDRESS FROM STD
WHERE ADDRESS IN ('KTM', 'BIKE', 'PTN');

BETWEEN

SELECT SALARY FROM EMP
WHERE SALARY BETWEEN 3000 AND 40000;

PERCENTAGE

SELECT NAME FROM STD
WHERE NAME LIKE '%a';

UNDERSCORE

SELECT NAME FROM STD
WHERE NAME LIKE '%__B';

HAVING

SELECT NAME FROM STD
WHERE GRADE = 'A'
HAVING MARKS > 800;
